# License Scope: Useful Examples and Qilin Framework

## Purpose

This directory mixes reusable framework code, third-party quantum simulation
code, historical samples, and game/content examples. It is not a single
license scope.

## Ownership and intended license

Only `qilin-game-framework/` components expressly listed in its own
`LICENSE_SCOPE.md` enter the Apache-2.0 code scope. All other files and
subdirectories here are excluded unless a more specific license or scope file
states otherwise.

MicroQiskit remains under its upstream license and notices. Standalone
examples, missions, levels, artwork, audio, and game experiences remain
reserved pending ownership and provenance review.

## Contributions

Do not assume that placing a file under `0UsefulExamples/` makes it open
source. New examples must state separate code and content terms and document
all embedded third-party components.
